ScrollSpy

On this page

Introduction Features Pricing FAQ
Introduction

ScrollSpy automatically updates navigation links based on scroll position. It watches the page or container scroll and highlights the active section link in the sidebar navigation as you scroll through the content.

Features

Key features include automatic active state toggling, support for nested navigation, works with any scrollable container, and smooth highlight transitions as sections come into view.

Pricing

All Bootstrap components including ScrollSpy are completely free and open source under the MIT license. No subscription required — just link the CDN or install via npm and start building.

FAQ

Q: Does ScrollSpy require jQuery? No — Bootstrap 5 is pure vanilla JavaScript with no jQuery dependency. Q: Can I spy on a div instead of the body? Yes, just add the data attributes to any scrollable container element.
